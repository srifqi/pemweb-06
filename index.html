Permasalahan Float

Jika sebuah elemen memiliki anak yang semuanya punya properti float, elemen tersebut akan kehilangan isinya sehingga tinggi elemen tersebut akan nol.

Hal tersebut bisa dibetulkan dengan menggunakan properti clear.

Kanan
<div style="background: green; padding: 8px;">
    <div style="background: orange; float: left;">Kiri</div>
    <div style="background: orange; float: right;">Kanan</div>
    <div style="clear: both;"></div>
</div>

Hal ini tidak disarankan karena properti float tidak dibuat untuk membagi tata letak seperti di atas. Ada properti-properti lain yang lebih sesuai untuk itu (baca tentang CSS Flexbox).

Pseudo-Class pada CSS

Pseudo-class adalah perintah yang ditaruh setelah selector untuk memilih keadaan-keadaan khusus dari elemen yang terpilih.

Contoh tautan 
a:link { color: blue; }
a:visited { color: purple; }
a:hover { color: cyan; }
a:active { color: red; }
  • Anak pertama
  • Anak kedua
  • Anak ketiga
  • Anak keempat
  • Anak terakhir
li:first-child { color: red; }
li:nth-child(3) { color: darkorange; }
li:nth-child(2n) { text-decoration: underline; }
li:last-child { color: green; }

CSS Reset

Kita dapat mengatur ulang nilai bawaan dari peramban dengan menggunakan tanda bintang * sebagai selector semua elemen.

* {
    margin: 0;
    padding: 0;
    font-family: sans-serif;
}
